# Krystian Bua academic website

Jekyll/GitHub Pages website with four navigation items: About, Research, CV, Contact.

## Publish on GitHub Pages

1. Upload all files and folders to the repository root.
2. Delete `.nojekyll` if present.
3. In Settings > Pages, publish from the `main` branch and `/ (root)`.
4. Wait for the Pages build to finish.

The CV menu item links directly to the Google Drive PDF.
